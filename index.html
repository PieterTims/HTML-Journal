In the table of contents part of the page, there are links that, when clicked brings users to a different part of the page. this is done by using the <a> tag, then using an attribute called href to link to a different part of a page that has been named using id attribute. href is usually used to hyperlink to HTTP-based URLs, and other URL schemes supported by browsers, like images, emails, and telephones.

Tables are made using <table> to define the table:

Tag Description
<thead/tbody/tfooter> defines three different groups of contents.
<tr> defines table rows.
<th/td> defines table cells, <th> creates header style cells.
There is also <colgroup> and <col>, which are used for formatting each column in a table.
